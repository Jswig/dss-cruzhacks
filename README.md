The number one source for transparency, brah

- Use python3.7~
- Install [pipenv](https://github.com/pypa/pipenv)
- Run `pipenv install` in one of the app directories.

To run the server:

```
cd server
pipenv install
pipenv shell
export FLASK_ENV=development
flask run
```
